# 边际成本（Marginal Cost）

> 经济学概念

**定义**：每多生产/提供**一单位**产品或服务所增加的**总成本**。

**核心公式**： MC=ΔQΔTC​

**关键理解**：
- 不是平均成本，而是"增量成本"
- 通常呈U型曲线：先下降（规模效应）→ 后上升（资源瓶颈）
- **决策依据**：只要边际收益 > 边际成本，就应该继续生产

**实际例子**：

| 场景   | 边际成本表现                         |
| :--- | :----------------------------- |
| 航空公司 | 起飞后多载一名乘客的成本 ≈ 一杯饮料 + 一份餐食（极低） |
| 软件开发 | 多卖一份授权的成本 ≈ 0（复制成本）            |
| 制造业  | 加班生产需支付1.5倍工资，边际成本陡增           |

# 边际效应（Marginal Effect）

> 心理学/社会学概念

**定义**：投入与产出的**非线性关系**，强调"过犹不及"。

付出到达一定界限时，所带来的回报。

**经典例子**：
- **学习**：每天学8小时效果最佳，学到12小时效率反而下降（疲劳）
- **广告**：看3遍记住品牌，看30遍产生反感
- **激励**：奖金从1万→2万干劲十足，100万→101万无感

# 边际效用（Marginal Utility）

> 经济学概念

**定义**：每多消费**一单位**商品带来的**额外满足感**。

**规律**：边际效用递减 —— 吃得越饱，下一口饭的满足感越低。

**例子**：
- 第一杯奶茶：满足感 10分
- 第二杯奶茶：满足感 6分
- 第三杯奶茶：满足感 2分，甚至为负（撑了）
